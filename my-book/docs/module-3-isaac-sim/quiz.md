# Module 3 Quiz: Isaac Sim

Test your understanding of Isaac Sim with this quiz. Each question has one correct answer unless specified otherwise.

## Question 1: What is Isaac Sim built on?
A) Unity Engine
B) Unreal Engine
C) Omniverse Platform
D) ROS Platform

<details>
<summary>Click to reveal answer</summary>

**Answer: C) Omniverse Platform**

Isaac Sim is built on NVIDIA's Omniverse platform, which provides real-time collaboration, USD (Universal Scene Description) support, and advanced rendering capabilities.

</details>

## Question 2: Which physics engine does Isaac Sim use?
A) ODE (Open Dynamics Engine)
B) NVIDIA PhysX 5.0
C) Bullet Physics
D) Havok Physics

<details>
<summary>Click to reveal answer</summary>

**Answer: B) NVIDIA PhysX 5.0**

Isaac Sim uses PhysX 5.0, NVIDIA's state-of-the-art physics engine, which provides accurate simulation for robot dynamics and realistic contact modeling.

## Question 4: True or False: Isaac Sim uses USD (Universal Scene Description) as its native scene format.

B) False

## Question 5: What is domain randomization used for in Isaac Sim?
A) To make simulations run faster
B) To vary environment parameters and improve model robustness
C) To reduce computational requirements
D) To create more realistic graphics

<details>
<summary>Click to reveal answer</summary>

**Answer: B) To vary environment parameters and improve model robustness**

Domain randomization involves varying environment parameters (colors, lighting, physics) to train more robust AI systems that can handle real-world variations.

</details>

## Question 11: What is curriculum learning in the context of Isaac Sim?
A) Learning basic concepts before advanced ones
B) Gradually increasing task difficulty during training
C) Following a structured learning path
D) Using multiple learning algorithms

<details>
<summary>Click to reveal answer</summary>

**Answer: B) Gradually increasing task difficulty during training**

</details>

## Question 12: What is the primary purpose of sim-to-real transfer techniques?
A) To make simulation faster
B) To ensure behaviors learned in simulation work on real robots
C) To reduce computational requirements
D) To improve graphics quality

<details>
<summary>Click to reveal answer</summary>

**Answer: B) To ensure behaviors learned in simulation work on real robots**

Sim-to-real transfer techniques aim to bridge the "reality gap" and ensure that robot behaviors and AI models trained in simulation can successfully operate on physical robots.

</details>
